Download data

There are several ways to access the data in the database.

By a dedicated download app

The app is found under “Insights >> Perovskite Database Project” in the left side menu in MaterialsZone’s system found on the link below (see instructions here on how to obtain access to the interactive graphics)

Link to interactive graphics

Fill out the form to download and to access the interactive graphics (for new users)

Sign in to download and view the interactive graphics (for existing users)

The app provides four types of data download.

  1. Download all data in the database in the form of a .csv file
  2. Download all data associated to a specified DOI number
  3. Download data based on interactive filtering
  4. Download data based on a list of database ID-numbers

From the interactive graphics

For each graph generated from the interactive tools, the data in that graph can be downloaded as a .csv file.
